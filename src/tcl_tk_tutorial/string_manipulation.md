# 字符串操作

**String Manipulation**

构造了一些数学脚本后，咱们移步到字符串。字符串，顾名思义，就是一串字符，a string of characters。现在，咱们来试着构造一些脚本，熟悉字符串的用法。


将字符串赋值给变量，与将数字赋值给变量一样简单。

```tcl
set str "This is a string"
```

咱们可以在其中加入一些特殊字符，以格式化输出结果。

```tcl
set str "Tcl/tk\nis\na\ngood\nlanguage"
```

这将给到变量 `str` 以下文本：

```console
Tcl/tk
is
a
good
language
```
